<h2 id="usage" > 👷 Usage </h2>
Required! Install git, node and yarn (or npm).

```bash
# Clone Repository
$ git clone https://github.com/RodrigoSaantos/sevn-esportes.git

# Go to folder classroom
$ cd sevn/

# Install Dependencies
$ yarn
# or
$ npm install

# Run Application
$ yarn dev
# or
$ npm run dev

# Access localhost
http://localhost:5173/
```
